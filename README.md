# Responsive Student Management Portal

## Project Overview

The Responsive Student Management Portal is a web application developed using HTML, CSS, and Bootstrap 5. It provides an easy-to-use interface for managing student information, attendance records, academic results, and communication.

The portal is fully responsive and works smoothly on mobile, tablet, and desktop devices.

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
